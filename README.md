1️⃣ What is the difference between var, let, and const?

answer: Difference between var, let, and const
var is old style, works inside functions, can be used again and again.
let works in blocks, you can change it but not reuse the same name in same block.
const also block-based, can’t change or reuse, for things that stay the same.

2️⃣ What is the spread operator (...)?

Answer :Spread operator (...)
The three dots ... are used to copy,  or open arrays/objects easily.


3️⃣ What is the difference between map(), filter(), and forEach()?

Answer : Difference between map(), filter(), and forEach()
map() makes a new array by changing each item.
filter() makes a new array only with items that pass some check.
forEach() just goes through items, doesn’t return anything.


4️⃣ What is an arrow function?

Answer : Arrow function
A short way to write functions with =>.
Saves typing and handles this automatically.



5️⃣ What are template literals?

Answer : Template literals
Use backticks ` instead of quotes.
Can add variables and write multiple lines easily.
